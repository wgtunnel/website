        <div className="platform-section">
            <h2>Stable Release</h2>

            > ⚠️ **Requirements**
            >
            > Only **systemd-based Linux distributions** are supported.
            > The firewall must use **nftables** or **iptables with the nft backend (`iptables-nft`)**.

            <h3>Debian</h3>

            <div className="download-container">
                <OutlinedIconButton
                    onClick={() => downloadLatestAsset('https://api.github.com/repos/wgtunnel/desktop/releases/latest', /^wgtunnel_.*_amd64\.deb$/i)}
                    label="Download .deb Package"
                    icon={FaLinux}
                />
            </div>

            **Install**
            ```bash
            sudo apt install ./wgtunnel*.deb
            ```

            <h3>Arch Linux</h3>

            <p>
                For Arch, the app is available on the [AUR](https://aur.archlinux.org/packages/wgtunnel-bin).
            You can install it using an AUR helper such as `yay`:
            </p>

            **Install**
            ```bash
            yay -S wgtunnel-bin
            ```

            **Start Daemon**
            ```bash
            sudo systemctl enable --now wgtunnel-daemon.service
            ```

            <h3>Tarball Installation</h3>

            <div className="download-container">
                <OutlinedIconButton
                    onClick={() => downloadLatestAsset('https://api.github.com/repos/wgtunnel/desktop/releases/latest', /^wgtunnel-.*-linux-amd64\.tar\.gz$/i)}
                    label="Download Tarball"
                    icon={FaLinux}
                />
            </div>

            <p>From the directory where you downloaded the file:</p>

            **Install**
            ```bash
            tar -xzf wgtunnel-*.tar.gz && \
            cd wgtunnel-*/ && \
            curl -LO https://raw.githubusercontent.com/wgtunnel/desktop/master/scripts/linux/install.sh && \
            chmod +x install.sh && \
            ./install.sh
            ```

            <small>Optional uninstall script and full details available in the <a href="https://github.com/wgtunnel/desktop" target="_blank" rel="noopener noreferrer">desktop repository</a>.</small>
        </div>
